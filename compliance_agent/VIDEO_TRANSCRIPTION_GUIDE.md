# 视频转录功能说明

## 📋 如何确认视频文案是否被解析

现在桌面应用已经添加了**视频转录结果显示**功能，你可以清楚地看到视频文案是否被成功解析。

## 🎯 界面变化

### 新增区域：视频转录文本 (自动解析)

在"笔记文本内容"下方，新增了一个文本框，专门用于显示视频转录结果：

```
┌─────────────────────────────────────┐
│  笔记文本内容:                │
│  [文本输入框]                   │
└─────────────────────────────────────┘
┌─────────────────────────────────────┐
│  视频转录文本 (自动解析):      │  ← 新增
│  [转录结果显示框]               │
└─────────────────────────────────────┘
```

## ✅ 转录成功标识

当视频转录成功时，会显示：

```
✅ 视频转录成功！

[这里是视频中的语音内容...]
```

## ❌ 转录失败标识

当视频转录失败时，会显示：

```
❌ 视频转录失败或无语音内容

请检查视频文件是否包含语音内容
```

## 🔍 使用步骤

1. **选择视频文件** - 点击"选择视频"按钮
2. **（可选）输入笔记文本** - 补充视频中的文字描述
3. **点击"开始审核"** - 系统会自动：
   - 转录视频语音为文字
   - 在"视频转录文本"框中显示转录结果
   - 将转录文本与笔记文本合并进行审核
4. **查看转录结果** - 检查"视频转录文本"框中的内容

## 📝 测试建议

### 测试1：有语音的视频
- 选择一个包含中文语音的视频文件
- 点击"开始审核"
- 等待转录完成
- 查看"视频转录文本"框是否显示转录内容

### 测试2：无语音的视频
- 选择一个只有背景音乐或无声音的视频
- 点击"开始审核"
- 查看"视频转录文本"框是否显示失败提示

### 测试3：视频+文本
- 选择视频文件
- 在"笔记文本"框中输入补充文字
- 点击"开始审核"
- 查看审核结果中是否包含视频转录内容

## ⚙️ 技术说明

### Whisper 模型
- **模型大小**：base（默认）
- **支持语言**：中文（zh）
- **处理时间**：取决于视频长度
- **准确率**：通常 >95%

### 转录流程
```
视频文件
    ↓
Whisper 模型加载
    ↓
音频提取
    ↓
语音识别
    ↓
文本输出
    ↓
显示在界面
```

## 💡 常见问题

**Q: 为什么视频转录是空的？**
A: 可能原因：
- 视频没有语音内容
- 视频音频格式不支持
- Whisper 模型未正确加载

**Q: 转录内容不准确怎么办？**
A: 可以：
- 使用更大的 Whisper 模型（small/medium）
- 确保视频音频质量良好
- 手动在"笔记文本"框中补充正确内容

**Q: 转录需要多长时间？**
A: 取决于：
- 视频长度（约 1-2 分钟/分钟）
- 模型大小（base 模型最快）
- 电脑性能（GPU 会更快）
